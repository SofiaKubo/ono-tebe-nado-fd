# ono-tebe-nado-fd

A responsive landing page for an auction of misunderstood items.

## 🚀 Project Overview

This project was built during a frontend developer course to practice semantic HTML, Flexbox and Grid layout, and adaptive design. The design is based on a Figma mockup and includes multiple layout blocks, interactive elements, and responsive behavior.

## ✅ Key Features

* Semantic and accessible HTML structure
* BEM methodology for CSS class naming
* Flexbox and Grid used for layout
* Connected local fonts in correct order
* Responsive design without layout breaking
* Valid HTML and clean, readable CSS
* Fully matches the provided Figma design
* Overflow and margin collapsing issues fixed
* Horizontal scroll limited to card container only

## 🛠 Tech Stack

* HTML5
* CSS3 (Flexbox, Grid)
* No frameworks, vanilla CSS only
* Git + GitHub
